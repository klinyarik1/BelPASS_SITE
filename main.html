BelPASS

Первая отечественная аппаратно-программная платформа для комплексной защиты информации

Структура платформы

Актуальность

В 68% успешных кибератак в Беларуси используется социальная инженерия. Платформа BelPASS призвана исключить человеческий фактор из цепочки угроз.

"Цель — решение системной проблемы кибербезопасности в Беларуси через создание и внедрение первой отечественной аппаратно-программной платформы для комплексной защиты информации BelPASS."

Структура платформы

BelPASS H-series

Аппаратно-программный модуль

Устройство для управления паролями. Содержит в себе криптографические процедуры AES-256 и SHA-128.

BelPASS P-series

Программный модуль

Приложение для управления паролями. Предназначено для конфиденциального хранения учетных записей.

Edu Game

Обучающая игра

Обучающая игра для формирования базовых навыков по защите от угроз информационной безопасности
